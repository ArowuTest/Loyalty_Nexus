# Loyalty Nexus - API Documentation (V1)

## 🏛️ Ingestor API
`POST /api/v1/recharge/ingest?msisdn=...&amount=...`
- accepts massive bursts of events from MTN Gateway.
- status: 202 Accepted.

## 👤 User API
`GET /api/v1/user/profile?msisdn=...`
- returns balance, tier, and streak status.

## 🎰 Spin Engine
`POST /api/v1/spin/play?msisdn=...`
- deducts 1 spin credit and returns prize result.

## 🎨 Nexus Studio
`GET /api/v1/studio/tools`
- list available creative tools and point costs.

`POST /api/v1/studio/chat`
- Ask Nexus conversational AI (Groq/Gemini fallback).

`POST /api/v1/studio/generate/image`
- Create category (FAL.AI/Flux).

`POST /api/v1/studio/generate/knowledge`
- Learn category (NotebookLM).

`POST /api/v1/studio/generate/build`
- Build category (Document Automation).

`GET /api/v1/studio/gallery?user_id=...`
- Retrieve generated asset history.

## 💳 Payment & Subs
